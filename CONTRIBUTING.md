# Working with This Repository

## For Participants

### Setup
1. **Fork this repository** (if working in teams)
2. Clone to your local machine
3. Create a virtual environment
4. Install dependencies: `pip install -r requirements.txt`

### Workflow
1. Create your analysis in a Jupyter notebook or Python scripts
2. Build your forecasting model
3. Create your dashboard
4. Generate predictions CSV
5. Prepare your presentation

### Submission
When ready to submit:
1. Ensure your predictions CSV matches the format in `example_submission_format.csv`
2. Make sure your code runs without errors
3. Test your dashboard works
4. Commit your work (if using Git)
5. Submit as instructed by organizers
